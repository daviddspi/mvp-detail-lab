# MVP Detail Lab - Premium Detailing Studio

![MVP Detail Lab Logo](/public/logo.png)

A high-end, modern web application for **MVP Detail Lab**, an exclusive auto-detailing studio located in Zemun, Belgrade. Built with a focus on sharp aesthetics, premium motion design, and seamless user experience.

## ✨ Key Features
- **High-Contrast Design**: Pure Black & White UI for a professional, architectural aesthetic.
- **Premium Motion**: Staggered entrance animations using Framer Motion.
- **Interactive Gallery**: Smart video playback and lightbox experience.
- **Responsive Detailing**: Fully optimized for mobile, tablet, and desktop.
- **Service Explorer**: Detailed breakdown of aesthetic and protective treatments.

## 🛠️ Technology Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: Vercel

## 🚀 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/daviddspi/mvp-detail-lab.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
